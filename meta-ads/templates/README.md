## Suggested workflow
1. **Research the audience** → `audience-research-BCOP.md` (avatar, UPSYD segment, interests).
2. **Engineer the hook & offer** → `hook-formulas.md` + `offer-frameworks.md`.
3. **Write the ad copy** → `ad-copy-building-blocks.md`, season with `13-elements-persuasive-copy.md`.
4. **Script the video** (Stage 3) → `video-script-EDIE.md`.
5. **Pre-publish gut check** → `7-power-questions.md`.

## Cross-cutting reminders
- **UPSYD** (Unaware → Problem → Solution → Your-Solution → Deal) underpins hook depth, copy length, and audience choice. Default start: Level 3 (Solution Aware).
- **Hook ≠ Offer.** One offer can have ten hooks.
- **Incompleteness Principle:** the offer must fulfill the hook's promise yet stay incomplete so the paid product delivers the bigger solution.
- **Match value/trust to the size of the ask.**
- Beginners: start with ONE offer and keep it simple.
