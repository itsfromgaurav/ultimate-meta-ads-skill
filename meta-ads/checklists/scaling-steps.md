# Scaling Steps Checklist

> The Five-Tier Scaling System (Ralph Burns) — step-by-step processes to grow small successful Meta (Facebook/Instagram) campaigns while holding target CPA. **Try each tier in order: 1 before 2, 2 before 3, etc.** Combining all five at once = Massive Rapid Scaling. Source: *Ultimate Guide to Facebook Advertising (3rd ed.)*.

> Prerequisite: pass the `pre-scale-readiness.md` checklist first.

---

## Tier 1 — Duplication Scaling

> Make exact copies of a well-performing ad set, keeping the **same starting budget** and the **same Post ID**.
> When to use: CPA is **$100 or lower**; ad set has **≥10 conversions** below target CPA; audience large, ideally **>1 million**.

- [ ] In Ads Manager, set calendar to **last seven days**
- [ ] Sort ad sets by CPA, smallest to largest
- [ ] Screen by volume of conversions; scroll to ad sets with significant counts; select all ads at or below your CPA
- [ ] Highlight those ads, click **Create Similar Ad Set**
- [ ] When the new window opens, review and press **Place Order**
- [ ] Search the auto-appended "-copy" and rename all with **"DUP" plus the date or a version number**
- [ ] Hit **Edit**, scroll to **Budget and Schedule**, enter the daily budget (the original starting budget)
- [ ] Select all duplicates, click **Enter Post ID**, insert the original ad's Post ID, press **Submit**
- [ ] Always duplicate the **ORIGINAL** ad, not the duplicate ad sets

## Tier 2 — Audience Scaling

> Broaden reach into new audiences.

- [ ] Duplicate your entire **Michigan Method** campaign
- [ ] Replace the original **20 interests with 20 new interests**, keeping the same creative
- [ ] (Research path) Use Audience Insights and/or the Power Editor Suggestion Tool to find 5–20 new relevant interests
- [ ] Verify reach stays in the **400K–2M** sweet spot
- [ ] Insert the **Post ID** to keep social proof

## Tier 3 — Alteration Scaling

> Like Duplication Scaling, but make **minor, data-driven** creative/copy tweaks for a fresh look to the same audience.

- [ ] Set calendar to **last seven days**
- [ ] Sort ad sets by CPA smallest to largest
- [ ] Screen by conversion volume for ad sets with significant conversions and lowest CPA
- [ ] Highlight those ads, click **Create Similar Ad Set**
- [ ] Enter new images or new copy (tweak examples: statement→question, add brackets/emojis, combine a winning image with another winner's copy, swap for a similar-themed image)
- [ ] Rename the ad to reflect the variation (e.g., `[name] Copy 7 Img 3`)
- [ ] Hit **Review Changes** and place order
- [ ] Keep audiences **large** so different ads aren't seen too frequently
- [ ] (Advanced) After duplicating the campaign, create **3 new creatives + 3 new ad copies (= 9 variations)**; keep the original 20 interests in the original Michigan Method campaign

## Tier 4 — Large Budget Scaling

> Duplicate winning ad sets into **NEW campaigns** and increase budget **5–20x** the original. Be extremely cautious.

- [ ] Duplicate your lowest-CPA ad sets and insert them into a new campaign
- [ ] Increase daily budget to your comfort level, ideally **2–3x target CPA**
- [ ] Start new campaigns in the **mornings** (avoid evenings — budget pacing skews results)
- [ ] (Ninja trick) When increasing budgets by large amounts, change the **lower end of the age range by one year** so Facebook treats it as a different audience
- [ ] Note: being more aggressive than 3x CPA works about **1 in 5 times** — only with a really hot offer

## Tier 5 — Small Budget Scaling

> Most conservative. Raise the budget **~20% every three days** until CPA exceeds target, then roll back.

- [ ] Set calendar to **last seven days**
- [ ] Sort ad sets by CPA smallest to largest
- [ ] Screen by conversion volume for ad sets with significant conversions and lowest CPA
- [ ] Click **Edit** and increase daily budget by **15–20%**; Save and close
- [ ] Check **every three days** that CPA is below target; if so, increase daily budget another **15–20%**
- [ ] When target CPA is exceeded, **roll back by at least two increments**
- [ ] Allow **72 hours** for Facebook to settle before judging a budget change
- [ ] Don't run this on **more than ~10 ad sets** (becomes unmanageable)

## Massive Rapid Scaling (combine all tiers simultaneously)

- [ ] Use **duplication** to add ad sets within the same campaign AND create a brand-new campaign
- [ ] Use **audience scaling** to broaden into new audiences
- [ ] Create more ads/ad sets/campaigns via **alteration scaling**
- [ ] Increase spend via **large budget scaling** — all at once

## Naming conventions

```
# Duplication Scaling — rename auto-created "-copy" ad sets:
DUP <date-or-version-number>

# Alteration Scaling — rename to reflect the variation:
[name] Copy 7 Img 3
```

## Key numbers / thresholds

| Metric | Value / threshold |
|---|---|
| Tier 1 CPA ceiling | ≤ $100 |
| "Performing well" conversions | ≥ 10 per ad set (below target CPA) |
| Ideal audience size | > 1 million |
| Tier 4 budget multiplier | 5x–20x original budget |
| Tier 4 recommended daily budget | 2x–3x target CPA |
| Tier 4 example ($25 CPA, $10/day orig) | $50–$75/day |
| Tier 4 example ($100 CPA) | $300/day (3x target CPA) |
| >3x CPA aggressiveness success rate | ~1 in 5 |
| Advertiser disclosure | $300/day → ~$3 cost per lead (amazing offers only) |
| Tier 5 increment & cadence | 15–20% every 3 days |
| Tier 5 roll-back when CPA exceeded | ≥ 2 increments |
| Facebook adjustment window | 72 hours (3 days) |
| Tier 5 manageability cap | ~10 ad sets |
| Best campaign launch time | Mornings (avoid evenings) |
