# Pre-Launch Checklist

> Everything that must be in place before any Meta (Facebook/Instagram) campaign goes live: account foundation, page completeness, objective selection, audience research, copy QA, and pre-flight strategy. Source: *Ultimate Guide to Facebook Advertising (3rd ed.)*.

---

## 1. Foundation before launching ads (Ch. 2 / Ch. 3)

Build these in order:

- [ ] Buy the new domain (e.g., GoDaddy)
- [ ] Install WordPress / build the site
- [ ] Build the landing pages
- [ ] Create the Facebook Page (business Page, **not** a personal profile)
- [ ] Create the Business Manager account
- [ ] Create the ad account
- [ ] Set up tracking (pixel / conversion tracking) — do ASAP after the ad account exists
- [ ] Create a lead magnet (one per business division, if applicable)
- [ ] Create and publish the campaign

## 2. Minimum prerequisites to run campaigns (Ch. 9)

- [ ] Facebook **Page** (business Page, not a personal profile)
- [ ] **Ad account**
- [ ] Your **unique selling system**

## 3. Page completeness before promoting (Ch. 9)

- [ ] Profile picture
- [ ] Cover photo
- [ ] Business description
- [ ] Username (searchable, not already taken)
- [ ] At least some posted content

> Pitfall: When creating the ad account, **time zone and currency are permanent** — set them correctly the first time (Ch. 9).
> Pitfall: Profiles have "Friends," Pages have "Likes" — never advertise from a personal Profile (Ch. 9).

## 4. Choose the objective BEFORE building (Ch. 26)

- [ ] Decide which of the three categories (Awareness / Consideration / Conversion) the ad serves
- [ ] Drill down to the specific objective within that category
- [ ] Confirm the objective matches your ACTUAL goal (don't pick Traffic/Engagement if you want sales)
- [ ] Choose the objective BEFORE building — you **cannot** change it on a live campaign
- [ ] If running a Conversions objective: confirm the Facebook pixel is installed
- [ ] If running a paid event: use Conversions, **not** Event Responses
- [ ] If two+ objectives could work and budget allows: plan a test
- [ ] After placing the order, remember the new campaign is **paused** — activate it (add more ad sets first if desired)

## 5. Pre-flight strategy diagnostics (Ch. 13, 14, Three-Step Video Ad Formula)

- [ ] Identify which of the three Business Categories you fall into and use that category's proven selling system
      (1. Retail/Local/Service; 2. eCommerce/Physical; 3. Digital/Information incl. SaaS)
- [ ] Map your target audience across the UPSYD awareness levels (Unaware → Problem → Solution → Your-Solution → Deal)
- [ ] Decide where on the UPSYD staircase to start (default: **Level 3, Solution Aware**)
- [ ] Match the value/trust in your ad to the size of the ask
- [ ] Ask: "How is that person coming into our sales funnel?" (guard up vs guard down)
- [ ] Run the 7 Power Questions on the copy before finishing the ad (see `ad-copy-qa.md`)

## 6. Pre-publish copy check — 7 Power Questions (Ch. 20)

- [ ] Does the offer appear novel, unique, and distinctive?
- [ ] Is the offer simple and easy to understand?
- [ ] Is there an opening question that triggers consistency/action?
- [ ] Have I pre-exposed the audience to a desired emotional concept?
- [ ] Have I tapped mental links/associations (memory, nostalgia, identity)?
- [ ] Have I opened a loop to hold attention until the close?
- [ ] Have I created a mental/visual portal to new opportunity?

## 7. Ad-to-landing-page congruency check / Ad Scent (Ch. 16 / Troubleshooting)

- [ ] Landing-page headline matches the ad's hook
- [ ] Ad copy and landing-page copy are aligned ("ad scent")
- [ ] Color schemes match to reinforce congruency
- [ ] Ad and landing page clearly look like they belong together

## 8. Always use a thank-you page (Ch. 12)

- [ ] Build a thank-you page (Facebook no longer keeps coupon redemption on-platform)
- [ ] For giveaways, place a coupon/offer on the thank-you page to filter lead quality

## Key numbers / thresholds

| Item | Value |
|---|---|
| UPSYD default starting level | Level 3 (Solution Aware) |
| Landing-page conversion rate (average reference) | ~20% |
| Landing-page conversion rate (problem level) | < 10% |
| New campaign state on creation | Paused (must activate) |
