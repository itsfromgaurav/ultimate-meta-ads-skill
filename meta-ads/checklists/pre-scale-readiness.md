# Pre-Scale Readiness Checklist

> The confidence bar an ad set must clear before scaling, plus the foundation and housekeeping that must be in place. Source: *Ultimate Guide to Facebook Advertising (3rd ed.)*, Five-Tier Scaling System / Ch. 14–20.

---

## 1. Pre-scale readiness (per ad set)

> Try the scaling tiers only on ad sets that pass all of these.

- [ ] Ad set has at least **10 conversions**
- [ ] CPA is below your desired target
- [ ] You have enough data to feel confident
- [ ] (Duplication/Alteration) Audience is large, ideally **>1 million**

## 2. Foundation must be in place first (Ch. 16–20)

- [ ] First two Flight Plan stages executed correctly (the foundation for massive scale and long-term ROI)
- [ ] You have already passed the **1,000+ conversions** milestone at an acceptable cost/ROI (Stage Three entry)
- [ ] Hooks and offers have already been tested and validated
- [ ] You are working from data, not guesses (sort by CPA smallest→largest, screen by conversion volume)

## 3. Conversion-volume floor for the algorithm

- [ ] Large audiences are fed at least **20–30 conversions/week per ad set** (bare minimum)
- [ ] Preferred: **10–20+ conversions/DAY** fed to Facebook for optimal learning
- [ ] For smaller-population countries, audiences are **100K–500K** instead of 400K–2M

## 4. Advanced scaling housekeeping

- [ ] Duplication Scaling: always duplicate the **ORIGINAL** ad, not the duplicate ad sets
- [ ] Duplicate at the **original** starting budget, not the later-raised one
- [ ] Audience Scaling: duplicate the entire Michigan Method campaign, then swap the original **20 interests** for **20 new interests**, keeping the same creative
- [ ] Alteration Scaling: after duplicating the campaign, create **3 new creatives + 3 new ad copies (= 9 variations)**; keep the original 20 interests in the original campaign
- [ ] Reuse the original **Post ID** in all duplicates to consolidate social proof

## 5. Tier order is respected

- [ ] You will try the methods **in order** — Tier 1 before 2, 2 before 3, etc. (don't jump straight to aggressive budget scaling)

---

## Key numbers / thresholds

| Metric | Value / threshold |
|---|---|
| "Performing well" conversions | ≥ 10 per ad set (below target CPA) |
| Duplication Scaling CPA ceiling | ≤ $100 |
| Ideal audience size (Duplication & Alteration) | > 1 million |
| Min conversions per ad set (large audiences) | 20–30/week (ideally 10–20+/day) |
| Audience sweet spot (US) | 400K–2M |
| Audience sweet spot (smaller countries) | 100K–500K |
| Stage Three entry | 1,000+ conversions at acceptable cost/ROI |
| First hard milestone | 100+ conversions (hardest part of the journey) |
