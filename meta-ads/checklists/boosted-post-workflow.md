# Boosted Post Workflow Checklist

> Facebook's "Easy Button": how to boost a post and run Dennis Yu's dollar-a-day test-and-scale + switch-boost amplification system. Source: *Ultimate Guide to Facebook Advertising (3rd ed.)*, "Facebook Easy Button: Boosted Posts" / "More Boosted Posts".

---

## 1. How to Boost a Post (the four steps)

- [ ] Add content to your page
- [ ] Click the big blue **"Boost Post"** button
- [ ] Select your audience
- [ ] Assign a budget

## 2. Dollar-a-day test-and-scale workflow

- [ ] Publish posts organically and measure organic results **first**
- [ ] Stack worthy posts against each other by boosting each at **$1/day for 7 days** (or test each at **$50** — Larry Kim's method)
- [ ] Compare reach and engagement across the boosted posts
- [ ] Identify the winners ("unicorns"); ignore the "donkeys"
- [ ] Put more spend behind the winners
- [ ] Switch-boost winners to strategic / influencer (media inception) audiences, preserving social proof
- [ ] Create "unicorn babies": switch audience, duplicate the ad set against a second audience, and/or make derivative content
- [ ] Add winners to a "greatest hits" list and re-boost evergreen ones (look back over the last **24 months**)

## 3. Pitfalls to avoid

- [ ] Don't boost losers (boosting multiplies results — validate organically first)
- [ ] Don't chase a daily "moonshot" — cultivate proven unicorns instead
- [ ] Don't treat the Boost button as a substitute for fundamentals
- [ ] Don't treat boosting as one-and-done — re-boost and switch-boost evergreen winners

## Key numbers / thresholds

| Item | Value |
|---|---|
| Dennis Yu minimal boost budget | $1/day for 7 days |
| Larry Kim per-post test budget | $50 per post |
| Evergreen look-back window | Last 24 months |
