# Meta Ads Checklists

Standalone, runnable checklists extracted from the 15 reference files in `../references/`. Every checklist preserves the exact numbers, thresholds, and step ordering from *Ultimate Guide to Facebook Advertising (3rd ed.)*. Use them as live "pre-flight" and operating checklists — copy a file, tick boxes as you go.

## Suggested order of operations

1. `pre-launch.md` (foundation + objective + strategy)
2. `pixel-tracking-setup.md` (tracking in place ASAP)
3. `funnel-offer-build.md` (hook, offer, funnel)
4. `audience-build.md` (research & build targeting)
5. `ad-copy-qa.md` + `creative-qa.md` (QA before publishing)
6. `boosted-post-workflow.md` (optional fast/cheap testing path)
7. `troubleshooting-diagnostic.md` (when results disappoint)
8. `pre-scale-readiness.md` → `scaling-steps.md` (once proven)

All thresholds (e.g., ≥10 conversions before scaling, Tier 1 CPA ≤ $100, 400K–2M audience sweet spot, ~$1/day × 7 boost tests, 15–20% every-3-days budget steps, 72-hour adjustment window) are carried verbatim from the references.
